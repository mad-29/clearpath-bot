# Clearpath Credit Services — AI Customer Service Chatbot

A customer service chatbot built for an Australian credit services company, powered by the Anthropic Claude API.

## Live Demo
🔗 https://clearpath-bot-olive.vercel.app

## What it does
- 💳 **Payment Plans** — collects customer details and sets up a payment arrangement
- 🤝 **Hardship Applications** — handles customers experiencing financial difficulty with empathy and refers to support services
- ⚠️ **Disputes & Complaints** — manages debt disputes and escalates to human agents when needed
- Follows Australian debt collection compliance guidelines (ACCC, AFCA)

## Tech Stack
- React
- Anthropic Claude API
- Vercel (deployment)

## Australian Compliance Features
- References Financial Counselling Australia (1800 007 007)
- Informs customers of AFCA complaint rights
- Follows ACCC debt collection guidelines
- Never uses threatening or misleading language

## Run Locally
```bash
git clone https://github.com/mad-29/clearpath-bot.git
cd clearpath-bot
npm install
npm start
```
